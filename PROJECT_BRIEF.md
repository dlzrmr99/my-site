# Personal Website 项目说明

创建日期：2026-04-25

## 项目目标

这个目录用于逐步建立一个专属个人网站。用户目前以前端开发 0 基础为起点，目标不是一次性生成完整网站，而是在学习过程中逐步理解 HTML、CSS、JavaScript、版本管理和网站部署，并把这些知识实际用在个人网站上。

## Codex 的长期任务

后续在这个目录工作时，Codex 需要充当教学者和协作开发者：

1. 先解释当前要学的概念，再带用户完成一小步可运行的网站改动。
2. 每次修改网站前，说明要改什么、为什么这样改、涉及哪些文件。
3. 保持改动小而清晰，适合 0 基础学习者理解。
4. 避免一次性堆叠复杂框架；优先使用 HTML、CSS、JavaScript 基础能力。
5. 每次完成后更新学习进度文档，记录完成内容、学到的知识、下一步任务。
6. 尊重用户已有内容，不随意重写用户写过的文字、页面和 Git 变更。

## 当前起点

当前目录已有：

- `index.html`：个人网站首页的最小 HTML 页面。
- `README.md`：项目说明的早期草稿。
- `docs/learning-plan.md`：学习路线和计划表。
- `docs/progress.md`：学习进度记录。

## 推荐工作方式

每次学习和开发按这个顺序进行：

1. 确认本次目标。
2. 用简单语言解释相关概念。
3. 修改或新增少量代码。
4. 运行或打开页面检查效果。
5. 总结本次学到的内容。
6. 更新 `docs/progress.md`。

## 网站内容方向

网站会从最简单的个人首页开始，逐步加入：

- 个人简介
- 技能或兴趣
- 项目作品
- 学习记录
- 联系方式
- 响应式布局
- 简单交互
- 部署上线
